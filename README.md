# BrainIn Manifest Importer

BrainIn Manifest Importer is a small Chrome extension that helps import a BrainIn deployment manifest into the BrainIn program administration form.

The extension reads a manifest generated by the BrainIn DevTools Unity package and fills supported parts of the BrainIn form, such as input parameters and partial result definitions.

## Installation

1. Click **Code**.
2. Click **Download ZIP**.
3. Extract the downloaded ZIP file.
4. Open Chrome and go to:

   ```text
   chrome://extensions
   ```

5. Enable **Developer mode**.
6. Click **Load unpacked**.
7. Select the extracted extension folder.

The extension should now be available in Chrome.

## Usage

1. Open a BrainIn program administration page:

   ```text
   https://brainin.kiv.zcu.cz/Program/Manage/<ProgramId>
   ```

2. Select a deployment manifest JSON file.
3. Click **Import manifest into form**.
4. Review the imported values.
5. Save the BrainIn form manually.
